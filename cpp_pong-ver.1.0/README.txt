This game was made in C++ in (a bit more than) a week, using Visual Studio Community 2022.

I've never programmed anything in C++ before this personal challenge and I mostly followed this guy's tutorial:
https://www.youtube.com/playlist?list=PL7Ej6SUky135IAAR3PFCFyiVwanauRqj3

Explaining gamemodes:

GAMEMODE A: Single-player versus AI (a pretty dumb one).

GAMEMODE B: 2-player mode. 
Player 1 moves using W to go up and S to go down.
Player 2 moves using UP to go up and DOWN to go down.

Both gamemodes are essentially endless, so enjoy yourself(s).

To quit the game, close the window. To change gamemode, close the window and open the game again to select another mode.
(I didn't have time and patience to do it another way lol)
